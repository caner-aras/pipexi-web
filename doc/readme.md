# Connecteam Web

Modern web application built with the latest React ecosystem.

## Tech Stack

- **Next.js 16** (App Router)
- **React 19**
- **TypeScript**
- **Tailwind CSS v4**
- **shadcn/ui**
- **Lucide React** (Icons)
- **Geist** (Typography)
- **ESLint**

## UI

- shadcn/ui (Nova preset)
- Responsive design
- Accessible components
- Modern SaaS-inspired interface

## Development

```bash
npm install
npm run dev
```

## Project Goals

- Clean architecture
- Reusable UI components
- Type-safe development
- Modern React patterns
- Scalable codebase
- Performance-first approach
